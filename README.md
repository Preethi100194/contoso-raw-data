# Fabric Medallion Retail — End-to-End DP-600 Portfolio Project

An end-to-end data platform on Microsoft Fabric implementing the medallion
architecture (Bronze → Silver → Gold), semantic modeling in DirectLake, a
Power BI report, and a Dev → Test → Prod deployment pipeline.

Built as a hands-on study project for the **DP-600: Fabric Analytics Engineer**
certification.
